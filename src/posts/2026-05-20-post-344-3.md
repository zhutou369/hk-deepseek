## 行銷經理必學嘅 DeepSeek 提示詞黃金公式：R.T.C.F

經過大量實踐同優化，我總結出一個對行銷人特別有效嘅 DeepSeek 提示詞黃金公式，我叫佢做 **R.T.C.F. (Role, Task, Context/Constraints, Format)**。只要你喺落指令嗰陣，諗清楚並提供齊呢四個元素，DeepSeek 生成嘅內容質素就會有質嘅飛躍。

### 1. 角色 (Role)：畀AI一個身份

首先，你要為 DeepSeek 設定一個具體嘅「角色」。當AI扮演一個特定嘅專業角色時，佢會以該角色嘅視角、知識同語氣嚟回應，令輸出更貼合專業要求。

**點解重要：** 提升專業度同相關性。
**常見行銷角色例子：**
*   資深行銷顧問
*   社交媒體策略師
*   SEO專家
*   文案編輯
*   品牌經理
*   內容創作者

**範例：**
*   `你係一個資深嘅行銷顧問，專為香港嘅餐飲品牌提供數位行銷策略。`
*   `請你扮演一位專精於社群媒體操作嘅香港品牌經理。`

### 2. 任務 (Task)：明確你嘅目標同指令

其次，你需要清晰、具體咁闡述你希望 DeepSeek 完成嘅「任務」。避免模糊不清嘅指令，愈明確愈好。

**點解重要：** 確保AI理解你嘅核心需求，避免偏離主題。
**常見行銷任務例子：**
*   撰寫一篇部落格文章
*   生成五個社交媒體貼文嘅標題
*   分析競爭對手嘅SEO策略
*   為新產品設計一份電子郵件行銷文案
*   提供三個 Google Ads 廣告標題建議

**範例：**
*   `我需要你撰寫一篇關於「香港智能家居科技」嘅部落格文章。`
*   `請你為我新推出嘅健康飲品，設計五個 Instagram 嘅廣告文案。`

### 3. 背景/限制 (Context/Constraints)：提供必要資訊同設限

呢個係黃金公式中最關鍵嘅部分，提供足夠嘅「背景資訊」同「限制條件」會大大影響輸出質素。背景資訊包括品牌調性、目標受眾、產品特點、市場狀況等；限制條件則可能係字數、語氣、關鍵字、禁止出現嘅內容等。

**點解重要：** 確保內容嘅準確性、相關性、品牌一致性同符合特定規範。
**常見背景/限制例子：**
*   **品牌：** 品牌名稱、品牌故事、品牌價值、品牌形象（年輕活潑、專業權威）。
*   **目標受眾：** 年齡、性別、興趣、痛點、消費能力。
*   **產品/服務：** 特點、優勢、獨特賣點（USP）。
*   **市場：** 香港本地市場特色、當前趨勢。
*   **語氣：** 幽默、嚴肅、友善、促銷性。
*   **技術要求：** 包含特定關鍵字、避免某些詞語、字數限制、段落數。

**範例：**
*   `我哋嘅品牌叫「綠活港」，主打天然有機食材，目標客戶係注重健康、25-45歲嘅香港職業女性。語氣要親切、有溫度，同時具備專業知識。`
*   `文章需要包含「環保」、「本地農業」同「健康飲食」三個關鍵字，字數限制喺800至1000字。`
*   `請避免使用過於學術嘅詞彙，務求淺白易明。`

### 4. 格式 (Format)：指定輸出結構

最後，明確你希望DeepSeek以咩嘢「格式」輸出內容。呢個可以係 Markdown 格式、列表、表格、JSON 格式，或者特定嘅文案結構。

**點解重要：** 方便你後續嘅編輯同使用，提升效率。
**常見格式例子：**
*   Markdown 格式（標題、列表、粗體字）
*   分點列表 (Bullet Points)
*   表格 (Table)
*   JSON 格式（用於數據結構化）
*   短文案（含標題、正文、行動呼籲CTA）
*   五個標題同五句描述

**範例：**
*   `請以 Markdown 格式輸出，文章開頭要有 H2 標題，內文用 H3 分段，並用分點列表列出優勢。`
*   `請用表格形式，列出三款產品嘅特點、價格同目標受眾。`
*   `輸出應該包括一個主要標題、一段約150字嘅正文，以及一個清晰嘅行動呼籲（CTA）。`

### 黃金公式總結：`[角色] + [任務] + [背景/限制] + [格式]`

掌握咗呢個黃金公式，你就可以好似砌積木咁，將你嘅行銷需求精準地轉化為 DeepSeek 能夠理解並高效執行嘅指令。

## DeepSeek提示詞進階技巧與陷阱

掌握 R.T.C.F. 基礎黃金公式只係第一步，要將 DeepSeek 嘅潛能發揮到極致，仲需要一啲進階技巧。

### 1. 迭代優化：從錯誤中學習
提示詞工程唔係一次到位嘅事。好似優化網頁SEO咁，你需要不斷測試、調整同優化你嘅提示詞。如果第一次生成嘅內容唔滿意，唔使驚，回顧你嘅 R.T.C.F. 元素，睇下邊度可以再清晰啲、再具體啲。

*   **小步快跑：** 每次只修改一個元素，睇下對輸出有咩影響。
*   **具體反饋：** 如果唔滿意，明確講俾 DeepSeek 聽邊度唔好，例如「語氣太生硬，請更活潑啲」、「呢度資訊唔夠詳細，請補充」。

### 2. 鏈式提示 (Chain Prompting)：將複雜任務拆解
對於比較複雜嘅行銷項目，一步到位嘅提示詞往往好難做到最好。你可以將一個大任務拆解成幾個細小、循序漸進嘅子任務。

**例子：** 撰寫一份詳細嘅行銷計劃書
1.  **第一步：** 請求 DeepSeek 生成一份市場分析報告提綱。
2.  **第二步：** 根據提綱，請求 DeepSeek 撰寫報告嘅某一部分（例如「競品分析」）。
3.  **第三步：** 請求 DeepSeek 根據市場分析結果，提出3個目標受眾細分建議。
4.  **第四步：** 依此類推，逐步完成整個計劃書。

### 3. 少樣本學習 (Few-shot Learning)：畀AI睇範例
如果你對輸出嘅風格、結構有特定要求，可以直接喺提示詞中提供一至幾個高質素嘅範例。DeepSeek 會學習呢啲範例嘅模式同語氣，然後嘗試模仿。

**例子：** 希望生成嘅社交媒體貼文帶有特定幽默感。
`你係一個社交媒體文案專家。請你參考以下範例嘅幽默感同語氣，為我哋嘅寵物用品店撰寫一個新產品嘅宣傳貼文。`
`**範例1：** 「我哋嘅貓抓板，令主子愛不釋手，連貓奴嘅沙發都得救！🎉」`
`**範例2：** 「狗仔衝出街？唔使驚！我哋嘅防走失頸圈，比你媽咪睇實你更有效！」`

### 4. 負面提示 (Negative Prompting)：明確唔要咩
有時，直接話俾 DeepSeek 聽你唔想要咩，比單純講要咩更有效。

**例子：** 撰寫產品介紹，但避免使用「平」、「抵」等詞語，強調價值。
`請你為我哋嘅高端護膚品撰寫一份產品介紹，但請避免使用「便宜」、「折扣」、「特價」等詞彙。`

### 5. 安全與倫理考量：數據私隱同偏見
作為負責任嘅科技博主，我必須提醒大家喺使用大模型時，要留意網絡安全同倫理問題。

*   **數據私隱：** 除非 DeepSeek 有提供本地部署或者安全嘅私有化解決方案，否則切勿將敏感嘅客戶數據或企業內部機密信息直接輸入到公開嘅模型中。永遠要將網絡安全擺喺首位。
*   **偏見問題：** AI 模型係基於大量數據訓練而成，可能會繼承數據中存在嘅偏見。行銷經理需要仔細審查 DeepSeek 生成嘅內容，確保唔會產生歧視性、不實或具誤導性嘅信息。
*   **資訊核實：** AI 生成嘅內容唔係100%正確，特別係數據、事實類資訊。永遠要進行人工核實，確保資訊嘅準確性。

## DeepSeek 在香港中小企嘅數字轉型機遇
